Upload an SSH public key

Upload an SSH public key. This key will be provided to Aggregates when you create slivers. Some aggregates will install this key on reserved nodes, for use in logging in to your reserved resources.




Note: An SSH public key can be found in your $HOME/.ssh directory on Unix-based systems. Public keys will usually have a .pub extension.

Mac OS X Users: in the file chooser dialog, press Command-Shift-. (period) to display hidden directories so that the .ssh directory is visible.
