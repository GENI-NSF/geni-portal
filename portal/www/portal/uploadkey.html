Upload a public key

This page allows advanced users to upload an SSL public key. This key will be used to create a certificate associated with this account, usable by other tools (like Omni) for invoking GENI API calls.




Note: This is not an SSH public key, as generated by ssh-keygen. This is an SSL key. To generate a public key, you can use openssl as follows:
Generate an RSA keypair with a 1024 bit private key: 
$ openssl genrsa -out private_key.pem -des3 1024
Then extract the public key from that keypair: 
$ openssl rsa -pubout -in private_key.pem -out public_key.pem
A new file is created, public_key.pem, with the public key. Upload public_key.pem.
